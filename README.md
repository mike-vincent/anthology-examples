# Anthology Radio - Content Pipeline Examples

This repository contains examples of the content processing pipeline used by [Anthology Radio](https://apps.apple.com/app/anthology-radio/id6738029702), an iOS app for old-time radio drama.

## Pipeline Overview

Each episode goes through these stages:

1. **Logline** - A 2-3 sentence story summary (generated by fine-tuned GPT-4o)
2. **Transcript** - Full speech-to-text transcription (AssemblyAI)
3. **Treatment** - Narrative summary with character and plot details (GPT-4o)
4. **Outline** - CliffsNotes-style critical analysis (GPT-4o)
